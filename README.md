# 🎉 Bless - 随机温馨提示动画

一个使用 **React + Framer Motion** 实现的温馨提示弹出动画页面。
点击开始按钮后，会在屏幕中随机位置持续生成温馨小提示，自动淡入淡出，保持动态且温馨的视觉效果 💖。

---

## 🌟 功能简介

* 🪄 **随机弹出提示框**：
  每隔 400ms 随机生成一个新的提示框，位置完全随机，始终保持在屏幕范围内。

* 🔄 **自动替换旧提示**：
  页面上同时最多存在 30 个提示框，新的出现时旧的会自动消失。

* 🎨 **多彩背景与动画效果**：
  每个提示框都会从多种柔和的背景色中随机选择，并带有淡入淡出的动画。

* 🧘‍♀️ **温馨祝福语内置 80 条**：
  包含早安问候、天气提醒、鼓励语等，带上表情符号让提示更有温度。

## ⚙️ 参数说明

| 参数                | 含义          | 默认值        |
| ----------------- | ----------- | ---------- |
| `MAX_TIPS`        | 同时存在的提示数量上限 | `30`       |
| `INTERVAL`        | 新提示生成间隔（毫秒） | `400`      |
| `BOX_W` / `BOX_H` | 提示框宽高       | `250 / 60` |
| `bgColors`        | 随机背景颜色集合    | 柔和系 13 种   |
| `tips`            | 内置祝福语数组     | 80 条温馨提示   |

---

## 🧠 主要逻辑说明

1. **点击按钮后启动循环定时器**
   使用 `setInterval` 每隔 `INTERVAL` 毫秒创建一个新提示。

2. **生成随机位置与颜色**
   通过 `window.innerWidth`、`window.innerHeight` 控制生成的提示不超出屏幕。

3. **保持数量恒定**
   每次新增后判断长度是否超过 `MAX_TIPS`，超出则移除最早的一个。

4. **进出场动画控制**
   使用 `Framer Motion` 的 `<AnimatePresence>` 实现提示框淡入淡出效果。
